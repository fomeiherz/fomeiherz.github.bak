配置golang环境

配置本地仓库

本地新建一个仓库：E:\go\Repository\src
配置GOPATH路径为：E:\go\Repository（注意：不用配置 \src）

配置项目路径

本地新建项目路径：E:\work_golang\src
配置GOPATH路径为：E:\work_golang（注意：不用配置 \src）
作用：配置了该路径项目才可以引用本地的包。
